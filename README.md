# PBIG brandstyle build v4

Deze build bevat:
- het aangeleverde PBIG-logo als visueel anker
- de herkenbare zesdelige kleurstrook uit de huisstijl
- een donkerblauwe event/community uitstraling
- Segoe UI font stack via systeemfonts

Opmerking:
het aangeleverde .ttf-bestand is niet ingebed in deze distributie. De build gebruikt een Segoe UI stack waar beschikbaar op het apparaat.
