# PBIG Partner Lead Scanner - production frontend starter

Dit is een eerste productie-starter voor de PBIG partner scanner.

## Inhoud
- Next.js App Router
- Supabase browser/server clients
- login flow
- partner dashboard skeleton
- admin dashboard skeleton
- CSV export endpoint
- PBIG-achtige styling basis

## Snel starten
1. Kopieer `.env.example` naar `.env.local`
2. Vul de Supabase waarden in
3. `npm install`
4. `npm run dev`
